# Contributing to lazyjira

Thanks for your interest in contributing! 🎉

## Development Setup

```bash
git clone https://github.com/gotexis/lazyjira.git
cd lazyjira
pip install -e .
```

## Making Changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Run the tests: `python -m pytest`
4. Submit a pull request

## Code Style

- Pure Python stdlib — no external dependencies
- Type hints encouraged
- Keep it simple and readable

## Reporting Issues

- Use [GitHub Issues](https://github.com/gotexis/lazyjira/issues)
- Include your Python version and lazyjira version
- Include the full error output
